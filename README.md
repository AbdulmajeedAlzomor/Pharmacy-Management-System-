Pharmacy Management System – C# WinForms (.NET)

نظام إدارة صيدلية متكامل تم تطويره باستخدام C# Windows Forms، مع واجهات بسيطة ومنظمة تساعد الصيدليات على إدارة عملياتها اليومية بسهولة ودقة وكفاءة.
يعتمد النظام على واجهتين رئيسيتين:

واجهة المدير (Administrator Panel)

واجهة الصيدلي (Pharmacist Panel)



---

مميزات النظام

لوحة تحكم المدير – Admin Panel

إدارة المستخدمين: إضافة مستخدم جديد، تعديل بياناته، وعرض قائمة المستخدمين.

تحديث البيانات الشخصية وكلمة المرور.

تسجيل دخول وخروج آمن.

تصميم واضح يساعد على التحكم الكامل بخصائص النظام.


لوحة تحكم الصيدلي – Pharmacist Panel

إضافة الأدوية الجديدة بكافة بياناتها: الاسم، رقم الدواء، تاريخ التصنيع، تاريخ الانتهاء، السعر، الكمية.

تعديل بيانات الأدوية.

عرض قائمة الأدوية.

بيع الأدوية وتحديث الكمية المتوفرة.

تنبيهات خاصة بتاريخ انتهاء الدواء.

واجهة سهلة الاستخدام تساعد الصيدلي في أداء مهامه اليومية بسرعة ودقة.



---

التقنيات المستخدمة

C#

Windows Forms

SQL Server (إن وجد)

.NET Framework / .NET

أهداف المشروع

تسهيل إدارة الأدوية داخل الصيدلية.

تحسين دقة العمليات وتقليل الأخطاء البشرية.

تسريع الوصول إلى بيانات الأدوية والمستخدمين.

توفير بيئة عمل واضحة وآمنة لكل من المدير والصيدلي.
